# Jira-clone

Полнофункциональное веб-приложение для управления проектами и задачами, разработанное по аналогии с Jira.

## Описание

Jira-clone позволяет командам эффективно планировать работу, отслеживать задачи и управлять проектами. Приложение включает систему аутентификации, управление проектами с ролевой моделью, доску задач и личный профиль пользователя.

## Основной функционал

- Регистрация и аутентификация пользователей
- Создание и управление проектами
- Система ролей: Owner (владелец), Manager (менеджер), Member (участник)
- Создание задач с названием, описанием, статусом, приоритетом и сроком выполнения (due date)
- Назначение исполнителей на задачи
- Доска задач с группировкой по статусам (To Do, In Progress, Done)
- Комментарии к задачам
- Приглашение участников в проекты
- Просмотр личных задач на главной странице
- Управление личным профилем

## Технологии

### Бэкенд
- **Nest.js** — Node.js фреймворк
- **TypeORM** — ORM для работы с базой данных
- **JWT** — аутентификация через токены
- **TypeScript** — типизированный JavaScript

### Фронтенд
- **HTML5** — разметка страниц
- **CSS3** — стилизация (градиенты, анимации, адаптивность)
- **Vanilla JavaScript** — клиентская логика
- **Fetch API** — асинхронные запросы

## Требования

- Node.js (v18+)
- npm или yarn
- PostgreSQL (или другая СУБД)

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/JSpodavan/Jira-clone.git
cd jira-clone
```

2. Установите зависимости:
```bash
npm install
```

3. Настройте переменные окружения:
```bash
cp .env.example .env
```
Откройте `.env` и заполните свои значения

4. Запустите приложение:
```bash
npm run start:dev
```

5. Откройте в браузере: `http://localhost:3000/index.html`

## Структура проекта

```
src/
├── auth-module/          # Модуль аутентификации
├── projects-module/      # Модуль управления проектами
├── task/                 # Модуль управления задачами
└── entities/             # Сущности базы данных

public/                   # Фронтенд файлы
├── index.html           # Главная страница
├── projects.html        # Страница проектов
├── project.html         # Страница проекта
├── profile.html         # Профиль пользователя
└── reg.html             # Регистрация/вход
```
